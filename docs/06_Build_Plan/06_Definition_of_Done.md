# Definition of Done

The Dump Cup build is considered complete when:

## Admin

- All forms create, edit, and delete data correctly
- Match handicap logic verified with test cases
- Trade effective dating works correctly
- Tournament points roll up correctly
- No console errors
- RLS correctly enforced

## Data

- Views used for all public queries
- No direct public table access
- Schema snapshot updated
- Views snapshot updated

## Public UI

- Matches Figma layout
- Responsive on iPhone
- Background + overlay consistent
- No layout shift
- Data loads correctly
- Loading states handled

## Performance

- No unnecessary re-renders
- Queries limited to needed columns
- Proper indexing in database

## Documentation

- Schema snapshot updated
- Views snapshot updated
- Decisions log updated
- Build plan updated if necessary
