# View Strategy

Heavy calculations should be done in SQL views.

Recommended Views:

v_current_season
v_team_totals
v_score_progression
v_player_stats_by_season
v_player_stats_all_time
v_last_match
v_current_rosters
v_undrafted_players

---

Why Views?

- Reduces client logic
- Ensures consistency
- Improves performance
- Makes UI simpler
