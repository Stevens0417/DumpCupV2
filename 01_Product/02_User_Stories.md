# User Stories

## Public User

As a viewer,
I want to see the current team scores,
So I know which team is winning.

As a viewer,
I want to see how the score has progressed over time,
So I understand season momentum.

As a viewer,
I want to see player stats,
So I know who is leading the MVP race.

As a viewer,
I want to see past seasons and winners,
So I understand the tournament history.

As a viewer,
I want to view tournament posters and results,
So I can relive past events.

As a viewer,
I want to browse gallery photos,
So I can see highlights from the season.

---

## Admin User

As an admin,
I want to enter match results quickly,
So standings update automatically.

As an admin,
I want match types to auto-calculate handicaps,
So scoring is consistent and error-free.

As an admin,
I want to override scores when necessary,
So edge cases are handled.

As an admin,
I want to trade players without losing historical integrity,
So team totals remain accurate.

As an admin,
I want to define tournament position points,
So different tournaments can have custom scoring.

As an admin,
I want to upload posters and gallery images,
So the public UI stays visually engaging.
