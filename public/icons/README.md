# App Icons

Place the following PNG icon files in this folder before deploying.
All icons should use the Dump Cup logo/branding on a dark background (#0E1A2B).

| File             | Size (px)   | Used for                              |
|------------------|-------------|---------------------------------------|
| icon-120.png     | 120×120     | iPhone home screen (older devices)    |
| icon-152.png     | 152×152     | iPad home screen                      |
| icon-167.png     | 167×167     | iPad Pro home screen                  |
| icon-180.png     | 180×180     | iPhone home screen (current devices)  |
| icon-192.png     | 192×192     | Android / PWA manifest                |
| icon-512.png     | 512×512     | Android / PWA manifest (high-res)     |
| icon-1024.png    | 1024×1024   | App Store / full-resolution source    |

## Tips
- Use a square canvas with no rounded corners — iOS applies the rounding automatically.
- Export as PNG with a transparent or #0E1A2B background.
- You can generate all sizes from icon-1024.png using a tool like:
  https://www.appicon.co  or  https://maskable.app
