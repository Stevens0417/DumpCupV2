# iOS Splash / Launch Screens

Place the following PNG splash screen files in this folder.
They are shown while the PWA is loading on iOS (Safari "Add to Home Screen").

Use a dark background (#0E1A2B) with a centered Dump Cup logo.

| File                      | Dimensions (px) | Device                              |
|---------------------------|-----------------|-------------------------------------|
| splash-750x1334.png       | 750×1334        | iPhone SE / iPhone 8                |
| splash-1170x2532.png      | 1170×2532       | iPhone 12 / 13 / 14                 |
| splash-1284x2778.png      | 1284×2778       | iPhone 12 Pro Max / 13 Pro Max      |

## Tips
- Background color: #0E1A2B
- Center the logo vertically and horizontally.
- Keep the logo at roughly 30–40% of the screen width.
- A comprehensive set of sizes for all iOS devices can be generated at:
  https://progressier.com/pwa-icons-and-ios-splash-screen-generator
